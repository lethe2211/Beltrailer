概要説明：
学園祭の一環で作成した，Windows用のアクションクイズゲーム"クイズ！ ベルトレーラー β版"です．
Microsoft Visual C++ 2010上で，C++を用いて実装しました．
また，その際に，"DXライブラリ( http://homepage2.nifty.com/natupaji/DxLib/ )"というライブラリを使用させていただき，DirectX関連の処理を行っております．
ゲームの実行ファイルは，"\move_quiz\Release\move_quiz.exe"です．
また，ソースコード等は，"\move_quiz\move_quiz"以下に入っております．
操作方法は，
1. タイトル画面からZキーで問題選択画面に移動する．
2. 矢印キーとZキーで好きな問題を選択する．
3. 問題の正解となる箱が来たら，Zキーで選択する．問題の正解ではない箱が来たら，何もキーを押さず待つ．
4. 箱が通り過ぎると，その判断が正しかったかどうかが判定される．正解なら10点，不正解なら-10点．
5. これを繰り返し，最初に与えられたノルマクリアを目指す．
となっております．
ゲームの遊び方については，"\move_quiz\Release\README.txt"にも記述があります．
学園祭向けのゲームなので，ジョーク要素もあったりしますが，ご容赦ください．
